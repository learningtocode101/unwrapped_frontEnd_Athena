## User Stories
```
As an eco-conscious individual 
I would like an app that allows me to find the best and nearest stores  
that sell food with little or no plastic packaging so I can limit my use of single use plastic  

As an eco-conscious individual  
I would like an app that allows me to learn more about best eco-friendly and recycling practices  

As an eco-conscious business  
I would like an app that allows me to add my shop details  
so others know what food/products I sell with little or no plastic packaging   

As a developer 
I would like to add a sign up feature 
so businesses/individuals can sign up to add naked food/product shops details securely  

As a developer 
I would like to add a log in feature 
so businesses/individuals can log in to edit/update naked food shops details securely  
```
